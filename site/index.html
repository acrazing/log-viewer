Built for the tabs developers actually open

Stop pasting logs into random online formatters.

API responses

Turn raw JSON and nested error payloads into highlighted, foldable output without leaving the page.

CI output

Render ANSI terminal colors from build logs, test output, and copied command-line traces.

Config files

Auto-format YAML, Markdown, SQL, patches, diffs, XML, JavaScript, CSS, and HTML when Chrome opens them as text.

Format coverage

Readable views for the messy formats.

Fast from anywhere

Use the source you already have.

Log Viewer works from selected text, hovered DOM nodes, clipboard content, full page source, and page HTML. Recent entries stay available in a local History panel.

vvJSON from selection or hovered node
ppJSON from clipboard
ccAuto-detect current page source
xxANSI view for page text
ffToggle full-screen mode

Local by default

Your logs stay in your browser.

Formatting and history run locally. Log Viewer does not collect logs, send them to a server, sell data, or track your content.

  • Open source and MIT licensed
  • Local IndexedDB history
  • No server-side formatter
  • No analytics inside the extension
