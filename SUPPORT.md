# Support

RawLens is an open-source project maintained in public on GitHub.

## Where To Get Help

- Bugs: <https://github.com/RawLens/rawlens/issues/new?template=bug_report.yml>
- Feature requests: <https://github.com/RawLens/rawlens/issues/new?template=feature_request.yml>
- Chrome Web Store: <https://chromewebstore.google.com/detail/rawlens/lbnkfmnolbefifdccejjijdgdipnfaib>
- Website: <https://rawlens.github.io/rawlens/>

## Before Opening an Issue

Please check:

- You are using the latest Chrome Web Store version or the latest `master` build.
- The issue still happens after reloading the extension.
- The sample data is redacted or synthetic.

Useful details:

- RawLens version.
- Chrome version and operating system.
- Input source, such as selection, hovered node, clipboard, raw page, page source, page HTML, or History.
- Content type, file extension, or URL shape.
- A small sample that reproduces the issue.

## Security Issues

Do not report security vulnerabilities in public issues. Follow [SECURITY.md](./SECURITY.md).

## Support Expectations

There is no guaranteed response time. Clear reproductions and small examples are much easier to triage.
